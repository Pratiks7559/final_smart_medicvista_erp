PHARMAMGMT SIDEBAR - KEYBOARD SHORTCUTS SUMMARY
================================================================================

📋 GENERAL SHORTCUTS:
--------------------
Alt + L          → Logout


🛒 PURCHASE SECTION:
--------------------
Alt + P          → Purchase Challan
Alt + P + R      → Purchase Return
Alt + S + U      → Suppliers

Ctrl + P + I     → New Purchase Invoice
Ctrl + P + L     → Purchase Invoice List
Ctrl + P + Y     → Purchase Payment


💰 SALES SECTION:
-----------------
Alt + S          → Sales Challan
Alt + S + R      → Sales Return
Alt + C          → Customers
Alt + I          → Stock Issues

Ctrl + S + I     → New Sales Invoice (Quick Invoice)
Ctrl + S + L     → Sales Invoice List
Ctrl + S + Y     → Sales Receipt


📦 INVENTORY SECTION:
---------------------
Ctrl + I + P     → Product List
Ctrl + I + A     → All Products Inventory
Ctrl + I + 2     → All Products Inventory 2 (Real-time)
Ctrl + I + B     → Batch-wise Report
Ctrl + I + D     → Date-wise Report
Ctrl + I + S     → Stock Statement


⚙️ ADMIN SECTION (Admin Only):
-------------------------------
Ctrl + A + D     → Dashboard
Ctrl + A + U     → Add User
Ctrl + A + P     → Pharmacy Details
Ctrl + A + B     → Database Backup
Ctrl + A + C     → Contra Entry


================================================================================
SHORTCUT PATTERNS:
================================================================================

1. SINGLE KEY SHORTCUTS (Alt + Key):
   - Alt + L → Logout
   - Alt + P → Purchase Challan
   - Alt + S → Sales Challan
   - Alt + C → Customers
   - Alt + I → Stock Issues

2. SEQUENCE SHORTCUTS (Alt + Key1 + Key2):
   - Alt + P + R → Purchase Return
   - Alt + S + U → Suppliers
   - Alt + S + R → Sales Return

3. CTRL SHORTCUTS (Ctrl + Section + Action):
   
   PURCHASE (Ctrl + P + ...):
   - I → New Invoice
   - L → Invoice List
   - Y → Payment
   
   SALES (Ctrl + S + ...):
   - I → New Invoice
   - L → Invoice List
   - Y → Receipt
   
   INVENTORY (Ctrl + I + ...):
   - P → Product List
   - A → All Inventory
   - 2 → All Inventory 2 (Real-time)
   - B → Batch Report
   - D → Date Report
   - S → Stock Statement
   
   ADMIN (Ctrl + A + ...):
   - D → Dashboard
   - U → Add User
   - P → Pharmacy Details
   - B → Backup
   - C → Contra Entry

TIPS:
-----
1. All Alt shortcuts are single or double key combinations
2. All Ctrl shortcuts follow pattern: Ctrl + Section + Action
3. Sequence shortcuts have 800-1000ms timeout
4. Shortcuts work from any page in the application
5. No need to focus on sidebar - shortcuts work globally
